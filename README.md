# Cool Website

A modern React landing page built with Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this project to GitHub.
2. Go to Vercel and choose **Add New Project**.
3. Import the GitHub repository.
4. Use these settings:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`
5. Click **Deploy**.
